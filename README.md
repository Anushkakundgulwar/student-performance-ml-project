# student-performance-ml-project

##  About This Project
This project is about predicting how a student will perform academically and whether they will get placed or not. It uses simple machine learning models based on student data like study hours, attendance, skills, and backlogs.

##  What I Did in This Project
- Used Linear Regression to predict student percentage
- Used Logistic Regression to predict placement (Yes/No)
- Compared results using accuracy and error
- Plotted graphs to understand data better

## Data Used
- Study Hours
- Attendance
- Skills Score
- Backlogs
- Percentage
- Placement (0 or 1)

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## How to Run This Project
- Install required libraries using pip
- Run the Python file in your system or Colab
- See predictions and graphs
